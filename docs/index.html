Open source · Windows-first

보여줄 때만 연결되는
원격지원.

Remote Support Platform은 명시적 사용자 동의, 세션별 권한, 암호화된 P2P 전송을 중심에 둔 Windows 원격지원 프로젝트입니다.

베타 다운로드 소스 보기 ↗

v0.9.0-beta.1 · Windows x64 · 현재 개발용 바이너리는 서명되지 않았습니다.

명시적 호스트 동의세션별 임시 키직접 연결 + TURN콘텐츠 비저장
Built for real support

필요한 기능은 가깝게.
위험한 권한은 멀리.

휴대용 호스트와 설치형 운영자 콘솔이 화면 공유부터 입력, 클립보드, 파일과 채팅까지 승인된 범위 안에서만 동작합니다.

01

빠른 연결

호스트는 설치 없이 Agent를 실행하고 짧은 지원 코드를 전달합니다.

02

보이는 동의

운영자와 조직, 요청 권한을 확인한 뒤 필요한 범위만 승인합니다.

03

안전한 제어

화면 보기, 포인터, 키보드, 파일 전송 권한을 독립적으로 취소할 수 있습니다.

04

인터넷 연결

직접 P2P를 우선하고, 불가능할 때 인증된 TURN 경로로 자동 전환합니다.

05

데이터 기능

텍스트 클립보드, 채팅과 무결성 검증 파일 전송을 제공합니다.

06

운영 준비

SBOM, provenance, 감사 체인과 재현 가능한 패키지 검증을 포함합니다.

Consent-first security

원격 접속이 아니라
검증된 세션을 만듭니다.

  1. 1
    운영자 인증

    OIDC 기반 로그인으로 운영자와 테넌트 신원을 확인합니다.

  2. 2
    호스트 서명 동의

    승인 내용은 호스트의 임시 키로 서명되어 세션에 묶입니다.

  3. 3
    전송 종단 결합

    DTLS 지문과 승인된 피어를 상호 검증한 뒤 콘텐츠 채널을 엽니다.

  4. 4
    즉시 범위 축소

    호스트 취소와 긴급 종료는 진행 중 입력과 전송에도 즉시 적용됩니다.
