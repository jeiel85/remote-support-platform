# Definition of Done

A production feature is done only when:

- requirements and threat model are updated;
- implementation respects module and privilege boundaries;
- unit, integration and failure tests exist;
- authorization and tenant isolation are tested;
- telemetry and stable error codes are added;
- logs are reviewed for sensitive data;
- configuration and migration are versioned;
- accessibility/localization impact is handled;
- support and operations behavior is documented;
- dependency/license impact is reviewed;
- release gate evidence is attached;
- user-facing limitations are accurate.
